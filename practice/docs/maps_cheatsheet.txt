==========================================================
MAPS (MIPS subset) デバッガ コマンドチートシート
==========================================================

【起動方法】
  $ maps -i <source.s>
  指定したソースをアセンブル後、対話モードで起動

【終了方法】
  quit / exit / bye
  シミュレータの実行を終了する

----------------------------------------------------------
■ 実行制御 (Execution)
----------------------------------------------------------
  run <addr>
      指定アドレスから実行を開始する

  step <n>
      指定した n 個の命令をステップ実行する

  continue <addr>  (または cont)
      指定アドレスから実行を再開する

  reset
      シミュレータをリセットし、レジスタを初期化する

----------------------------------------------------------
■ ブレークポイント (Breakpoints)
----------------------------------------------------------
  breakpoint <addr>
      指定アドレスにブレークポイントを設定する

  delete <addr>
      指定アドレスのブレークポイントを削除する

  list
      設定されているすべてのブレークポイント一覧を表示する

----------------------------------------------------------
■ 状態確認 (Inspection)
----------------------------------------------------------
  regdump  (または print_all_regs)
      すべてのレジスタの状態（値）を表示する

  print <reg> | <addr>
      指定された「レジスタ」あるいは「メモリアドレス」の値を表示する

  memdump <addr>
      指定アドレスから始まるメモリの内容を表示する

  PrintSymbols
      定義されているシンボルとその値を表示する

----------------------------------------------------------
■ ファイル操作・その他 (File / Misc)
----------------------------------------------------------
  load <file>  (または asm)
      指定ファイルをアセンブルし、メモリにロードする

  help
      コマンド一覧と簡単な説明を表示する
