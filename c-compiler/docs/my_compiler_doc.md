# 概要

本プロジェクトは、情報工学実験C コンパイラ実験で作成したものである。
c言語ライクなプログラムをソース言語として、MIPSアーキテクチャのアセンブリ言語を目的言語とするコンパイラである。

## 実行方法

**必要なツール**

- gcc
- flex
- bison
- maps

**実行方法**

1. `make`
    コンパイラが作成されます。
2. テスト実行 

`make run_sum` `make run_kaizyo`
二つのテストが実行できます。`$a0`に結果が格納されているのが確認できます。

## サポート仕様

1. プログラムの構造
    変数宣言 + 文をプログラムとして認識します。片方だけだとプログラムとして認識しません。
2. 変数宣言
    `define x;` のように単一の変数の宣言のみ。
3. データ型
    `int`のみサポート。今後、配列もサポート予定。
4. 文
    - 代入文
    - ループ：whileループ
    - 式文：最終的な結果を返すために特別に定義。
5. 式
    - 四則演算
    - 整数
    - 変数
    - 括弧
6. 条件式
    - <

## コンパイラのアーキテクチャ

- c_program.l：字句解析を行うプログラムを作成。
- program.y：構文解析を行うプログラムを作成する。
- symbol_table.c: 変数の名前と場所を管理する。
- codegen.c：ASTをたどりながらアセンブリコードを生成していく。はじめと終わりの処理もサンプルコードを参考に作成。
- main.c：それぞれを呼び出し、コンパイラを生成。
